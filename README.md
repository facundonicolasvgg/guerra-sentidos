# Guerra-Sentidos

https://facundonicolasvgg.github.io/guerra-sentidos/

"El juego de cartas definitivo para salir de casa ya borrachos como una cuba. 'La Guerra de los Sentidos' es un juego de beber sin piedad que combina retos físicos absurdos, pruebas de reflejos y desafíos sensoriales donde mantener el equilibrio es casi imposible. Aquí no hay estrategia: o superas la prueba con los ojos vendados o te toca vaciar el vaso de un trago. Diseñado exclusivamente para destruir hígados, perder la vergüenza y calentar la noche a base de chupitazos."
