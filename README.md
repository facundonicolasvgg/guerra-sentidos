# Guerra-Sentidos

https://facundonicolasvgg.github.io/guerra-sentidos/

"Olvídate de los juegos de beber aburridos de siempre. Aquí vienes a ponerte fino filipino. 'La Guerra de los Sentidos' te propone un cara a cara donde tus reflejos se miden trago a trago. Supera desafíos ridículos usando el tacto, la vista y el oído para salvarte, porque si fallas tendras que lidiar con perder la ropa y beberte un chupito entre pecho y espalda. El último que quede en pie (y con ropa), gana."
